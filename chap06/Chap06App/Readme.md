## 6. 메소드로 코드 간추리기
> 타입에는 속성과 행위를 정의하는데, 속성은 C# 코드에서 필드로 표현되고, 행위는 **메서드(method)** 로 표현된다.
> C 같은 언어에서는 함수(function), 서브루틴(subroutine), 프로시저(procedure)라고도 한다.
> 메서드는 값을 반환하지 않는 함수를 정의할 수 있고, 전달하는 값과 반환받는 값의 자료형을 명시해야한다.
